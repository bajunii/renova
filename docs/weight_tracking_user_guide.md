# Weight Data Recording - User Guide

## Quick Start

### Step 1: Access Analytics Tab
1. Log in to your group organization account
2. Click on the **Analytics** button in the Quick Actions section
3. The Analytics overlay will open

### Step 2: Record Weight Data
1. Click the **"Record Weight Data"** button at the top
2. Fill in the form:
   - **Select EcoSpot**: Choose where the material was collected
   - **Material Type**: Select the type of material (Plastic, Paper, Glass, Metal, etc.)
   - **Weight (kg)**: Enter the weight in kilograms (e.g., 25.5)
   - **Notes** (optional): Add any additional information
3. Click **"Record"** to save

### Step 3: View Your Data

#### Statistics Cards
At the top of Analytics, you'll see four key metrics:
- **Total Weight**: All-time total weight collected
- **This Month**: Weight collected in current month
- **Total Records**: Number of weight entries
- **Avg Weight**: Average weight per record

#### Material Breakdown
Visual progress bars showing:
- Distribution across material types
- Total weight for each material
- Material icons for quick recognition

#### Recent Records
List of your last 10 weight entries showing:
- Material type with icon
- EcoSpot name
- Date and who recorded it
- Weight in kilograms

## Important Notes

### Before You Start
- **Create EcoSpots First**: You must have at least one EcoSpot created before you can record weight data
- If you try to record without an EcoSpot, you'll see a message to create one first

### Data Requirements
- **Weight**: Must be greater than 0 kg
- **EcoSpot**: Required - must select from dropdown
- **Material Type**: Required - select appropriate type
- **Notes**: Optional - for additional context

## Troubleshooting

### "Please create an EcoSpot first"
- You need to create at least one EcoSpot before recording weight
- Go to EcoSpots tab and click "Create New Ecospot"

### "Please enter a valid weight"
- Make sure weight is a positive number
- Use decimal point for precision (e.g., 25.5 not 25,5)
- Don't include units in the input (kg is added automatically)

### No records showing
- If you just recorded data, try closing and reopening Analytics
- Make sure you're logged in to the correct group account

### Data not saving
- Check your internet connection
- Verify you have permission to record data
- Try again in a few moments

## Best Practices

### Accuracy
- Weigh materials immediately after collection
- Use calibrated scales
- Record weight to at least 1 decimal place

### Consistency
- Record data daily for accurate trends
- Use consistent times (e.g., end of collection route)
- Always select the correct material type

### Documentation
- Use the Notes field for important details
- Record unusual collections or contamination
- Note any issues with materials

### Regular Review
- Check statistics weekly
- Compare monthly trends
- Identify high-performing EcoSpots
- Adjust collection strategies based on data

## Tips for Success

1. **Set a Schedule**: Record weight data at the same time each day
2. **Be Specific**: Select the most accurate material type
3. **Add Context**: Use notes for important observations
4. **Review Regularly**: Check your statistics weekly
5. **Share Data**: Use analytics for team meetings and reporting
6. **Set Goals**: Track progress toward collection targets
7. **Celebrate Wins**: Recognize milestones in total weight

## Privacy & Security

- All weight records are associated with your user account
- Only members of your group can view the data
- Records include timestamp and user name for accountability
- Data is securely stored in Firebase Firestore
